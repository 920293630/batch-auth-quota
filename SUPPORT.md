# 获取支持

如果你在使用 `batch-auth-quota` 时遇到问题，建议按下面方式分流：

## 适合提 Issue 的情况

### Bug 反馈

如果你确认是程序行为异常，请使用 `Bug Report` 模板，并尽量附上：

- 使用的命令
- 认证类型
- 配置来源（CLI / 环境变量 / `.env` / `config.json`）
- 关键报错信息
- 脱敏后的示例输出

### 功能建议

如果你希望增加新能力或优化交互，请使用 `Feature Request` 模板，并说明：

- 当前痛点
- 期望行为
- 预期收益

## 不建议直接提公开 Issue 的情况

以下情况建议先脱敏，或优先私下处理：

- 包含真实密钥
- 包含真实认证文件
- 包含可直接访问的内部地址
- 属于安全漏洞或敏感信息泄露问题

这类问题请先阅读 `SECURITY.md`。

## 使用问题排查顺序

在提问前，建议先自查：

1. 阅读 `README.md`
2. 阅读 `USAGE.md`
3. 检查 `.env`、环境变量与 `config.json` 是否冲突
4. 检查管理接口地址是否可用
5. 检查是否误把敏感数据直接带入命令或日志

## 当前维护范围

当前更优先处理：

- 明确可复现的 Bug
- 配置读取与统计口径问题
- 隔离/恢复流程问题
- 文档缺失或误导
- 低风险、聚焦的工程化改进

暂不优先：

- 与当前工具目标无关的大型功能扩展
- 无明确收益的大规模重构
- 无法复现的问题描述
